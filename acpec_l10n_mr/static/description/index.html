About the module

Ce module charge le modèle de plan comptable général de la Mauritanie. Nous fournissons un ensemble réduit de comptes et de taxes suffisant pour vous permettre de démarrer avec Odoo. Créez une base de données Odoo vide. Installez ce module, puis installez le module de comptabilité.

  • Cliquez sur configuration
  • Choisissez Taxe ou plan comptable


1. Mettre la TVA par défaut à 16%

2. Il offre un ensemble des comptes pour enregistrer les écritures comptables de la mauritanie
